分布式与集中式的人际网络

Posted on 2020-01-31 Edited on 2026-04-03 Word count in article: 574 Reading time ≈ 2 mins.

分布式是指，将一件综合性的事情，分布到多个地方，分别处理。评价：专一，稳定，无较强的依赖性。
集中式是指，将所有事情都堆积到一起，集中处理。评价：综合性较强，依赖性较高，稳定性较弱（一旦你所依赖的无法帮你很好的处理问题了，则整个问题将会坍塌。

不知何时，我与别人的交往从集中式的转换成了分布式的。

以前，我不管遇到什么问题基本上都会只想要也只能够跟同一个人或者两个人分享，但是到了现在，面对不同的情况，不同的心愿，我会选择去跟不同的人交流。比如说，我在数学上碰到了什么问题，我会去问谁，编程上碰到问题会问谁，关于对一些事情的思考我会想要去分享给谁，关于文学精神上的思考我会去找谁，关于学习上的感悟等又会向谁讲述，关于生活上的乐事，那些沙雕事我又会跟谁说等等很多。这里面所说的对象并不仅仅只是一两个人，而是三个四个乃至五个等若干人。

从稳固性的角度来看，显然后来发展的这种人际交往模式更加地稳固，分布式地解决问题，分布式地发展。思维更加开阔，看问题也更加地全面，不局限于自己，不局限于同两种模式，这种生活交流的模式我很喜欢。说其为何稳固，举一个物理上的例子，用一根柱子支撑一个木板和用多个柱子，三个四个，哪个更加稳固？支撑点更多，受力更加地均匀，自然得以长久。

只总跟一个人说会烦，且时间久了，你也会越发地想要去依赖于某个人。但是如果转变成多人的交往模式呢？就不一样了。你不会总从一个角度看问题，不会轻易地受其中某一个人行为的影响，情绪也会更加地稳定。

等等，我能够相出很多的优点。

哈，开心。

写作的方法

Posted on 2020-01-14 Edited on 2026-04-03 In 技巧 Word count in article: 1.4k Reading time ≈ 5 mins.

已经写了差不多有三年。一直写一直写，有一些心得，以此记下，以便参考。

写作的道理很简单，就是将你思考的过程写下来而已。

过程应该是怎样进行的，你就将这个做事的步骤写下来。思考的逻辑是怎样的，就将你进行思考的这一套线-逻辑写下来。比如说你在对一件事情发出感想的时候，引起你脑子进行这一系列的原因，引起思考的点，如何地使你得出这个结论的逻辑线写下来，就是一个很好的文章。

因为语言本就是一个过程/大脑思维活动的一个描述，一个代表，一个展现。

它终究是你脑子思考，手写下来的产物。

有人会说，啊，有的人瞎编瞎造也是写作啊，那种写作逻辑是怎样的呢？也是第二点。即将你瞎编瞎造脑子里面进行的活动记录下来，就形成了一篇文章或者是一个小说。一篇文章所进行的活动会少一些，连贯性没有那么强，只需进行一步写作就可以完成。但是小说所进行的活动就会多一些了。它需要你不断地思考，将前面一个故事详细地叙述，再写出如何引出来后面的一个故事，很难一步达到然后分多次地写出来，这是一个不连续思考的过程。要想写出一部很好的小说，在以后再次提笔写作时就必须要想出前面写了啥，后面怎样写才可以跟前面连贯起来等诸多问题。

上面主要讲述的是内容，但是这还不够，因为精彩的内容需要使用流畅多变的语言展现出来。如果你通篇都在说，我今天吃了一顿美味的饭菜，知道了原来世界上还有这么美味的饭菜啊，我觉得我也应该开始学习做饭了，因为我觉得吃好的饭菜真的能够带给人快乐，使人觉得特别的幸福。你吃饭嗨皮，开心，要学做饭，（脏话），谁不会说，三岁小孩都会说。多次使用“了”，“你觉得”，跟堆流水账一样没有任何吸引人想要继续读下去的欲望。

如何将极其出彩的内容用看起来丰富自然的语言展现出来呢？这需要多练，那如何在练习的过程中反复地打磨用来展现内容描述事情的语言呢？

遵循正确多变的原则，跟英语写作是一样的。

正确是最基本的要求。如果你不能够使用正确的语言，你怎么能够让读者顺利地了解到你所想传递的思想或者是让读者了解你所描绘的某个场景呢？怎样做？首先你要有一种能力，你能够读懂一篇文章。先理解自己是如何读懂的这篇文章，再用你自己的语言将你所读懂的这篇文章描述出出来。一个输入，一个输出，类似于费曼技巧。你能够读懂，说明你知晓了解了，你能清楚地讲述，表示你真正地理解了。达到第二步，你就可以进行写作了，因为你已经能够“正确地”用话来传输你的思想，因此也就很容易将这些话转化成书本上的文字进行表述让别人也能够理解你的思想和你想表达的某个想法或者意愿。你可以不断地跟别人讲话，讲述你现在的心情，介绍你刚看到的新闻，如果别人能够跟你在这一方面进行顺畅没有多少疑问的交流，就说明你真正正确地传达了你想传达的。这就是语言正确。

接着就是多变。

脑子不喜欢一成不变的文字，总是一个劲地多“了”很容易让人觉得枯燥无味，可以尝试在写作的过程中适当减少一句话最后一个字使用“了”的频率，将其用其他动词替换或者直接去掉。比如说，从这里我们就可以学习了，适当地看书是非常有必要的了，因为只有才能够…将了字去掉，就显得特别地干练。

大脑喜欢的是多变“好看”的文字。就我刚才所写到的一些句子进行解释，「你可以不断地跟别人讲话，讲述你现在的心情，介绍你刚看到的新闻，如果别人能够跟你在这一方面进行顺畅没有多少疑问的交流，就说明你真正正确地传达了你想传达的。这就是语言正确。」其实要是我以前写的话，就会使用三个讲述，在内容传达上面上看，完全没有问题，可是就是重复的次数多了，让人看着厌烦。所以在自己写作的时候，出现了这个问题可以尝试着像我这样进行修改，看书的过程中去看别人是怎样使用多变的文字来描述相类似的动作的。

将思考做事的过程完整的描述出来，再按照正确的语言和多变的文字的原则，这样多加练习，一定可以写出很好的文章。

这些就是我的一些写作的经验，希望能够给你/自己以后的写作尝试一份启迪。

2020考研总结-一次纪念

Posted on 2019-12-30 Edited on 2026-04-03 In 个人总结 Word count in article: 7k Reading time ≈ 25 mins.

概述：
考研主要是对四门学科的学习。按照其重要性来排序：数学、专业课、政治、英语。
考试具体科目：政治、英语二、数学三、统计学。

一、数学：

所花的时间最多，最后所获得的成就感也最多的一个科目。
分为基础、进阶、强化三个阶段。

基础：8-9.15。课本概念理解。

  • 用书：第六版高等教育出版社的上下《高等数学》、《线性代数》、概率则是用的直接开始进阶，用的考研辅导书王式安的《概率讲义》+华中科技大学出版社出版的《概率论与数理统计》。
  • 辅助: 即是课本概念的理解，定理的推导。
  • 具体措施：主要理解书本上面的概念，要求(1)对于定义，能够给别人用大白话讲清楚书本上的每个基本术语的定义，底线是能够给别人讲述出每个名词的定义是什么，如何运用，极限和导数的定义以及他们之间的关系。(2)之于定理，基本-中等难度的定理自己会证明，例如，各种中值定理的证明；线代秩的性质，他们的大小比较；概率的随机变量函数的概率密度函数是是怎样得到的（其实这个很简单，利用单调性，复合函数求导，密度函数非负的性质）。

进阶：9.15-10.15。课本概念运用，向考研进军。

  • 用书：高数，复习全书刷几遍（没记清）；线性代数：《线代讲义》刷两遍；概率继续前面的讲义。
  • 辅助：反复做旧题，总结思考，不断精简。
  • 具体措施：拿出白纸，不看答案做题，写出完整步骤。这一步的主要目的是规范化解题步骤+杜绝眼高手低+加强计算能力/技巧。已经会的题目，要删去，方便以后提高复习效率。

强化：10.15-12.20，考场模拟，真题+模拟题。

  • 套卷：真题04-07一遍 + 12-19刷两遍（第二遍一天3h-3.5h两套） +　2019李林六套卷/四套卷 +　2019合工大共创模拟卷1-3 +　2020张宇八套卷 + 2020李林六套卷/四套卷(3/4)
  • 辅助: 改错本。
  • 具体措施：时间：上午8:30-11:30准时卡点做，只可提前做完，不能延长考试时间，即便试卷难度很大。这是为了1适应考试时间。2模拟考试心态，即便有几道不会做的题目依旧要保证较高的分数。
  • 解题方略：一个小题，5分钟内没有思路放弃，一个大题8分钟之内没有思路，尝试写一些基本的步骤，得一些基础的分数；到考试只剩下半个小时的时间回去先检查选择填空，再接着是大题，保证会做的题目不因为粗心丢分。
  • 其他注意事项：在试题方面，将试题答案按照考试的空间写在A4纸上，将草稿纸用量控制在两张A4纸内（与考试对接，考场上一般没有多余的草稿纸）。上午考试尽量少喝水，一般考场是不允许带水的，有不一样的，这里统一定为考试不许带水。

更正复习：基本方法：早上做题，晚上花30min-1h时间改错。
详细要求：

  • 要求1.初期：改到改错本上，不可仅仅写在卷子上。并且分清楚错题是需要注重过程还是思路的，将易错点用红色/荧光笔重点标记出来，这是改错本层面的。
  • 要求2：更加重要的是：知识点的回顾，比如我经常会将第一类间断点和第二类间断点之间搞混淆，此时首先应该做的是回去看课本，将第一类间断点与第二类间断点的定理再次理解，并且搞清楚他们之间的区别，问自己是否可以举例说明，回顾全书，找几个类似的做一下。
  • 要求3：此时已经做了很多的题目了，已知三门学科的具体出题点在哪，这需要用新的改错本将错题按照错题类型进行归类，而不是试卷进行归类。详细措施：高数线代概率分开，高数分为（1）函数极限连续，（2）一元函数微分，（3）一元函数积分，（4）多元函数微积分，（5）级数，（6）微分方程。综合类的题目，将其归类到自己不会用/最重要的那一模块下。
  • 要求4：复习频率，至少两个星期一次。这我做的不好，经常给自己制定很多的任务，而最后完成的很累，量也不多。直到最后两个星期才开始将这些题目完完全全地归类整理。开始改错的时候搞一遍，这最后自己搞又搞一遍，划去已经掌握的题目，将仍旧经常犯错的题目思路留下进行整理。此外，还搞了一个自认为比较好的方法：将整理出来的错题按照章节进行统计，找出出错频率最高的进行最后一段时间的强化练习，最后发现我最容易出错的点居然是积分和导数的应用。

考前

  • 考前两个星期，将最近做题的易错点整理（特指低级错误）出来，进行纠正，防止考试的时候再犯相同的低级错误。
  • 这时，因为看过太多的错题，更正了很多的易错点，导致容易想当然的用自己已有的知识给出一个假设。以为题目完全就是我们所想象中的那个样子，容易犯很多低级错误，这时，需要在做题的时候注重将题目中的关键字画出来，而不是用自己的先验知识去解题。

做的跟之前不一样的：

  1. 从复习考研开始，数学一直都在学习，没有一天落下过。
  2. 将书本由厚变薄，做到了。复习全书，线代讲义，概统课本，随着复习过程的深入，一直在变薄，需要复习的东西也越来越少。
  3. 改错，确实将改错落实到位了。错题会改错，反思，下次不再错，更正错题以达到提高分数的目的。
  4. 后期的刷题一直在保证。每天上午8:30-11:30全部都是数学做题的时间，除去有事或者需要集中进行改错回顾外，没有一天没有做数学套题。

四、英语

英语则是单词+真题+作文。
简单分一下阶段

基础 单词＋阅读+作文背诵

背单词（不背单词考研核心词汇） + 英语一阅读热身（06-15） + 背诵 14篇作文

进阶 单词+阅读+英文写作（主攻大作文）

背单词 +　英语二阅读（10-19） + 记忆阅读文章中出现的新单词＋写作练习

强化 阅读+英文写作

英语二阅读（10-19滚动做，没有规律）+模仿真题作文+形成模板
写作练习（大作文）的具体措施：

  • 首先看视频，了解写作考点，写作要求，优秀作文的标准，高级单词、句子是怎样的。
  • 结合上一步的笔记，每天晚上都从中拿单词和句型，将他们放入到作文中进行演练。将所有学习到的句型都演练了一遍。用时一个月。
  • 融会贯通。抛却笔记，回归考研作文辅导书，写真题中的作文，先模仿真题中的句子，再结合自己的写作习惯进行写作，这是模仿阶段。模仿的有：由于对图表作文的第一段描述不够自信，每天从作文书中随意选取两篇进行第一段的仿写，两天再进行一次完整作文的仿写。这里还有小作文，也是模仿真题的作文，运用背诵到的句子，学习到的高级词汇、句型进行仿写。
  • 整理模板。将之前写的作文全部都整理到一起，标记出重点句型、词汇，挑明文章主旨的句子，句子之间的衔接词，阅读同时将其转化成语音听记忆。考前再写两篇（两大一小）。

政治：

  1. 考试的时候需要答得有条理一些。
  2. 背诵长段文字时，可根据Toy problems的思想，将文字拆解成一小段一小段，最后连成一整段进行整体背诵，记忆的会很牢固。也可以将需要背诵的内容录音，传入到喜马拉雅中去，走路睡觉都可以听。
  3. 刷1000有效。考前再刷1000题时，可以看题不做题，提高学习效率。看题不仅要看考察的知识点，还需要看错误选项是怎样设计的，找到自己经常容易选错的地方。之前我就经常会将一些用肯定语气描述的选项判定为错误。通过看题，可以有效培养“题感”，有效避免这种情况的发生。

英语

  1. 背诵可以参考政治。录音，拍照，一停下来就学习。
  2. 记住背诵作文的目的提高写作能力。从作文中学习优秀的句子，巧妙地运用到你的作文中。

数学

  1. 如果目标很高，在11月份一定要刷一刷计算量很大的套卷，例如张宇八套卷，今年或者往年的合工大的试题等，提高计算量。但是到了最后冲刺的时候就不用再这样做了，刷中等难度的习题保持手感即可。再最后复习改错的阶段，依旧不能眼高手低，对于经常算错的地方，一定要动手，在草稿纸上演算。
  2. 做题时间管理。选择填空，一个一般控制在5min左右，如果一个题目3-4min依旧没有思路，或者感觉计算量过大，先放下，等之后回过头来看。大题，一个一般16-17min，最多不能够超过20min。同选择填空，如果一个题目在7-8min一点思路都没有，赶紧跳到下一体，不用浪费时间。如果感觉有一点点的思路，但是不全，可以写一些上去，可以得一些步骤分（保证题目不会依旧可以多拿分）。
  3. 题目检测。坚持两个原则：1有十足把握会做的题目一次做好，不留时间后来检测；2最后至少预留20min的时间，回来检测。1是保证会做的一定做对，不让其可能出现错误，影响后来的做题状态。2是为了以防万一，在开始做题，对那些没有很大把握的题目（最好标注出来）进行检测，因为之前不会的，可能经过一轮的做题之后找到感觉，又知道怎么做了，可以将之前的不确定转化为确定。
  4. 草稿纸分配。考试的时候是一张大纸，总的表面积相当于两张A4纸，所以平时数学模拟训练时，平时模拟练习需要控制稿纸用量，不可以超过两张A4纸。
  5. 考前状态调整。考前的复习也就分为两个部分。一部分是复习经常容易忘记的公式性质定理。第二部分是错题的复习。包括思路不对的题目和计算错误的题目。思路不对的题目一般在进行改错的时候都总结到一起了，同时也要求你将自己考前一段时间经常出错的题目整理出来，一次性纠正，防止考试再出错。

  3. 学习精简题目。提高考前复习效率，稳定复习心态。

专业课

开始是按照笔试来准备的。这分了两个阶段。
分别是刷题 + 错题巩固 + 保持做题手感与速度

刷题是先将华科概率论与数理统计+辅导习题之前标记不会的刷了一遍，再去刷了一遍茆诗松的概率论与数理统计，做了网上找到的10套中山和中科大的初试题；
错题巩固，则是不断地看之前的错题；
做题保持手感，是去网上找的各个学校的概率论与数理统计的题目来做。这一步的主要目的是，练解题速度 + 计算速度。这个阶段每天要保持1.5h的计算。

说说我最后是怎么准备的吧。首先第一步，把所有面试可能问到的题目都过一遍，主要内容包括，概统、统计学原理、多元统计分析、时间序列分析和应用线性回归。
这一遍，对于每一个问题，要求你与你的同学一起将题目都讲述一遍。因为复试是需要你讲出来的，所以在进行模拟面试的时候也要注意练就你的表达能力。其中在回答题目的时候，只需要关注两点，一点内容，第二点是在进行内容阐释所使用的逻辑问题。还是跟之前英语练习的一样。在回答之前，需要使用5秒的时间，将问题的大致框架，即你打算按照怎样的套路来回答想一下，接着就需要你将具体的内容穿插进去，进行阐述了。

第二步，再过一遍，将相似的地方、就目前内容可以延申的点也补充上。考察发散思维以及前后联想能力。这需要你对前后的内容很熟悉。

第三步、问题抽查，检测，同时可以去找一些之前没有提及到的、很基本的问题来问对方，或者去网上找一些统计学简答题来考察对方。这样做的目的是查露补缺。

怎么准备？
首先是内容，接着是感悟。比如你参加数学建模竞赛，你就要首先搞明白竞赛的流程，论文是怎么写的，假设建立模型和模型修正等问题。接着需要说一下你是怎么解决问题的，这一步需要你去了解当初是怎么想到的这个解法。最后是对原解法的一个补充，当初你参加比赛的时候知识掌握的并不多，现如今你学习了更多知识那么你有没有想到什么好的方法来解决这个问题呢？这是一个很好的考察一个学生有没有探索精神的一个很好的方式。

毕业论文也是。先需要了解背景，你为什么选这个题目；再说你是怎么搞的；最后自己评价一下你的论文怎么样，比如其中使用了什么方法，它有什么优点缺点等，如何保证它的优点。
